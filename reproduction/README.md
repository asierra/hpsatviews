# Reproduction Kit — hpsatviews

This directory contains everything needed to reproduce the demonstration
outputs described in the accompanying paper.

## Requirements

| Dependency | Minimum version | Notes |
|---|---|---|
| GCC | 4.9 | Must support OpenMP (`-fopenmp`) |
| GNU Make | 3.81 | |
| libnetcdf | 4.6 | `libnetcdf-dev` on Debian/Ubuntu |
| libpng | 1.6 | `libpng-dev` on Debian/Ubuntu |
| GDAL | 3.0 | `libgdal-dev` on Debian/Ubuntu |
| curl | any | For downloading sample data |
| gdalinfo | 3.0 | For output validation (part of GDAL) |

Install on Debian/Ubuntu:
```bash
sudo apt install build-essential libnetcdf-dev libpng-dev libgdal-dev curl
```

## Steps

### 1. Build

From the project root:
```bash
make
```

### 2. Download sample data (~88 MB)

```bash
bash reproduction/download_sample.sh
```

This downloads 7 ABI channels from the NOAA public S3 bucket into `sample_data/`.
The scene is **GOES-16 CONUS, August 7 2024 at 13:00 UTC** (Day of Year 220).
At this time the solar terminator crosses the US Pacific coast (~120 °W), placing
most of the domain in full daylight — ideal for showcasing the Rayleigh atmospheric
correction — while the west coast is at dawn, enabling the day/night blend product.

No account or credentials are required; the NOAA S3 bucket is public.

### 3. Run the demo

```bash
bash reproduction/run_demo.sh
```

This script:
1. Recompiles the project from scratch.
2. Locates the downloaded anchor file automatically.
3. Produces three outputs in `reproduction/out/`:

| File | Product | Key feature demonstrated |
|---|---|---|
| `demo_truecolor.tif` | True Color RGB | Natural color baseline |
| `demo_truecolor_rayleigh.tif` | True Color + Rayleigh correction | Atmospheric scattering removal (bluer sky, improved contrast) |
| `demo_daynite.tif` | Day/Night composite | Automatic solar terminator blend with city lights |

4. Validates that output dimensions are consistent with the reference images in
   `expected_output/`.

## Expected output

The `expected_output/` directory contains 800-pixel-wide PNG thumbnails of the
reference outputs generated by the authors. Use them for visual comparison.

| Reference image | Product |
|---|---|
| `demo_truecolor.png` | True Color baseline |
| `demo_truecolor_rayleigh.png` | True Color with Rayleigh correction |
| `demo_daynite.png` | Day/Night composite |

## Notes

- The `sample_data/` directory and the generated GeoTIFFs in `out/` are listed
  in `.gitignore` and are not included in the repository.
- Full-resolution GeoTIFF outputs are approximately 23–26 MB each.
- Typical runtime on a 4-core machine: < 10 seconds per product.
